# SLAM-система для автономной навигации на бюджетных ARM-платформах

**Акцент на энергоэффективность и простоту использования**

Данный проект представляет собой прототип SLAM-системы (Simultaneous Localization and Mapping), предназначенной для работы на одноплатных компьютерах с поддержкой OpenCL, таких как Orange Pi 5. Основные компоненты: калибровка стереокамер, построение карты глубины в реальном времени, веб-интерфейс для управления и мониторинга.

Система разработана с учётом:
- минимальных требований к аппаратному обеспечению;
- использования OpenCL для ускорения вычислений на GPU;
- кросс-компиляции для ARM;
- возможности коммерческого использования по отдельной лицензии.

---

## Особенности

- **Калибровка стереопар** по шахматному шаблону с контролем качества (ошибка репроекции, покрытие ракурсов).
- **Построение карты глубины** на основе стереосопоставления, оптимизированное под OpenCL.
- **Веб-панель управления** (тонкий клиент) — доступ через браузер, отображение видеопотоков, тепловой карты глубины, телеметрии, журнала событий.
- **Кросс-компиляция** для ARM (Orange Pi) и нативная сборка для x86_64.
- **Двойное лицензирование**: GPLv3 для открытого сообщества и коммерческая лицензия для проприетарного использования.

## Требования

### Аппаратные
- Одноплатный компьютер с поддержкой OpenCL (Orange Pi 5, Raspberry Pi с GPU, x86_64 с любой видеокартой).
- Две USB-камеры с одинаковыми характеристиками, жёстко закреплённые в стереопаре.
- Wi-Fi или Ethernet для доступа к веб-интерфейсу.
- Устройство оператора с современным браузером (ПК, планшет, телефон).

### Программные
- ОС: Debian / Ubuntu (22.04 или новее) или Armbian.
- Компилятор: GCC 14+, CMake 4+, make.
- Зависимости: OpenCV (≥5.0), OpenCL (установленные драйверы), Eigen3.
- Для сборки под ARM: установленный кросс-компилятор (aarch64-linux-gnu-gcc) и Docker (опционально).

## Установка и сборка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-аккаунт/проект.git
   cd проект
   ```

2. Сборка под x86_64:
   ```bash
   mkdir build && cd build
   cmake ..
   make -j4
   ```

3. Сборка под ARM (Orange Pi) с использованием кросс-компиляции:
   ```bash
   ./build_arm.sh
   ```
   Скрипт автоматически использует кросс-компилятор и Docker для изолированной сборки.

После успешной сборки исполняемые файлы появятся в папке `build/bin/`.

## Использование

### Запуск сервера
На целевой платформе (Orange Pi) запустите все сервисы:
```bash
./ws_ctl
./calib
./cv
./capture
```

### Доступ к панели управления
Откройте в браузере адрес:
```
http://<ip-адрес-устройства>:9090
```

### Калибровка стереопары
1. На устройстве оператора отобразите шахматный шаблон (например, на планшете).
2. В веб-интерфейсе переключитесь в режим калибровки.
3. Плавно перемещайте стереопару, показывая шаблон под разными ракурсами.
4. Калибровка завершится автоматически при достижении стабильной ошибки репроекции.

### Построение карты глубины
После калибровки в веб-интерфейсе станет доступен режим «Тепловая карта глубины». Карта обновляется в реальном времени.
